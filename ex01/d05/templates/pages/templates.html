Django Template Engine

1. Template Blocks

Blocks define areas that child templates can override:

{% verbatim %}
{% block content %}
Default content
{% endblock %}
        {% endverbatim %}

2. For Loops

Iterate through lists and other iterables:

{% verbatim %}
{% for item in items %}
{{ item }}
{% empty %}
No items found
{% endfor %}
        {% endverbatim %}

3. If Statements

Conditional content display:

{% verbatim %}
{% if condition %}
Show this
{% elif other_condition %}
Show that
{% else %}
Default content
{% endif %}
        {% endverbatim %}

4. Context Variables

Display data passed from views:

{% verbatim %}
{{ variable }}
{{ object.attribute }}
{{ dictionary.key }}
        {% endverbatim %}
