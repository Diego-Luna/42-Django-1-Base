Static Page Display Process in Django

Step-by-Step Process

  1. HTTP Request

    User enters URL in browser (e.g., /ex01/display) which sends HTTP request to Django server

  2. URL Pattern Matching

    Django's URLconf (urls.py) matches the URL pattern to corresponding view function

  3. View Processing

    View function (e.g., display_view) processes the request and prepares context data

  4. Template Loading

    Django loads the template file (e.g., display.html) from templates directory

  5. Template Rendering

    Template engine processes template tags, filters and renders with context data

  6. HTTP Response

    Django sends the rendered HTML back to browser as HTTP response
